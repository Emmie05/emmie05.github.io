Interactive Resistor Color Code Generator and GPA Calculator

A toolkit with essential tools to calculate GPA and determine resistor values based on color codes.

Key Features

GPA Calculator

Calculate your GPA by entering courses, selecting grades, and inputting credit hours. The tool supports a wide range of grades and instantly provides your GPA.

Resistor Color Code Generator

Pick the colors for each resistor band and instantly get the calculated resistance value, making circuit design easier.

Responsive Design

Built with a mobile-first approach, ensuring that the toolkit works seamlessly across devices, from desktops to smartphones.

About the Project

This project was inspired by my academic journey in Electrical & Electronic Engineering. Often, I needed quick tools for GPA calculation and resistor value decoding, so I created this toolkit to solve those problems.
